Extreme Football Throwdown

Put extremefootballthrowdown and fretta13 folders in to garrysmod/gamemodes folder.
Add +gamemode extremefootballthrowdown before the +map declaration in your server startup script.

eft_ and xft_ maps are available here: http://steamcommunity.com/sharedfiles/filedetails/?id=244859331
and here: http://heavy.noxiousnet.com/downloadurl/maps

Uses fretta13, available at github.com/jetboom/fretta13 under the public domain. It has been included in this repository for convenience.
